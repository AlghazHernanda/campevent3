# GHZBlog Website

## There are various implementations on this website, namely:
- **Login and Registration Logic and Page** 
 
- **CRUD(Create, Read, Update, Delete) for Event **
   
- **Data Validation** 
   
- **Dashboard Admin Logic and Page** 

- **Authorization**
